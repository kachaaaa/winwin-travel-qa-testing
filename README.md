# WinWin Travel – QA Testing Project

This repository contains the results of manual testing performed as a test assignment for the WinWin Travel platform.

##  Scope
- Exploratory testing of landing page and search functionality
- Header component testing

##  Testing Types
- Functional testing
- UI/UX testing
- Validation testing
- Navigation testing

##  Bug Reports
A total of 6 bugs were identified, including:
- Registration error handling issue
- Email validation issue (whitespace handling)
- Broken navigation (404 page)
- Missing images in hotel listings
- UI layout issue (text overflow)
- Non-functional header features

## Test Cases
Test cases were created to verify the Header component, covering:
- Navigation elements
- User interactions
- Access control (unauthorized user behavior)

## Tools Used
- Manual testing
- Google Docs (Bug Reports)
- Google Sheets (Test Cases)

## Notes
This project was completed as a test assignment and is intended for demonstration purposes only.
